Tags <h1> a h6>

Função: São usadas para definir títulos HTML.

h1: Título Principal

h2: Subtítulo

h3: Tópico

h4: Subtópico

h5: Detalhe

h6: Observação

Tag <p>

Função: Define um páragrafo.

Exemplo: Este é um texto em um páragrafo.

Tag <small>

Função: Define textos menores (como direitos autorais e outros comentários paralelos).

Exemplo: Este é um texto pequeno.

Tag <i>

Função: Define uma parte do texto em um tom ou modo alternativo. O conteúdo interno normalmente é exibido em itálico.

Exemplo: Este texto está em "itálico".

Tag <u>

Função: Representa algum texto não articulado e com estilo diferente do texto normal, como palavras com erros ortográficos ou nomes próprios em texto chinês. O conteúdo interno normalmente é exibido com um sublinhado.

Exemplo: Este é um texto sublinhado.

Tag <strong>

Função: Usada para definir texto com grande importância. O conteúdo interno normalmente é exibido em negrito.

Exemplo: Este é um texto importante

Tags <ol> <ul> <li>

Função: Criar listas ordenadas e não ordenadas.

Exemplo de lista ordenada:

  1. Item 1
  2. Item 2
  3. Item 3

Exemplo de lista não ordenada:

  • Item A
  • Item B
  • Item C

Tag <a>

Função: Define um hiperlink, que é usado para vincular de uma página para outra.

Exemplo: Clique Aqui

Tag <hr>

Função: Define uma quebra temática em uma página HTML. Geralmente exibido como uma régua horizontal usada para separar conteúdo (ou definir uma alteração) em uma página.

Exemplo:

Tags <sub> <sup>

Função: Texto subscrito e sobrescrito

Exemplo: H 2 e 210

Tag <blockquote>

Função: Especifica uma seção que é citada de outra fonte.

Exemplo:

"Esta é uma citação"

Tag <del>

Função: Define o texto que foi excluído de um documento. Os navegadores geralmente riscam o texto excluído.

Exemplo: Texto antigo

Tag <abbr>

Função: Define uma abreviação ou um acrônimo, como "HTML", "CSS", "Sr.", "Dr.", "ASAP", "ATM".

Exemplo: HTML

Tag <font>

Função: Alterar a cor e tamanho do texto (tag antiga).

Exemplo: Texto colorido
