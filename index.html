Explorando as Tags do HTML

Tags <h1> <h2> <h3> <h4> <h5> <h6>

Função: Criar títulos e subtítulos com diferentes níveis de importância.

h1: Título Principal

h2: Subtítulo

h3: Tópico

h4: Subtópico

h5: Detalhe

h6: Observação

Tag <p>

Função: Criar parágrafos

Exemplo: Este é um parágrafo que explica um conceito

Tag <small>

Função: Mostrar texto menor, usado para observações

Exemplo: Este é um texto pequeno

Tag <i>

Função: Colocar texto em itálico

Exemplo: Este texto está em "itálico"

Tag <u>

Função: Sublinhado

Exemplo: Texto sublinhado

Tag <strong>

Função: Texto importante em negrito.

Exemplo: Texto importante

Tags <ol> <ul> <li>

Função: Criar listas ordenadas e não ordenadas.

Exemplo de lista ordenada:

  1. Item 1
  2. Item 2
  3. Item 3

Exemplo de lista não ordenada:

  • Item A
  • Item B
  • Item C
